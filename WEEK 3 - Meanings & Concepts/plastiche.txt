This poster by Piper Hughes is a pastiche in the sense that it uses similar techniques of drawing,
clearly inspired by Ed Fella’s work. It is a similar visual language that involves drawing by hand,
but the elements appears to be fragmented.

It is almost like it is a drawing of a collage of letters or things that look like letters. 

This is very similar to Ed Fella’s approach to design and illustration. You might say this is a similar visual language,
but a different idiom or dialect. Compositionally there is a general similarity in that it feels very uncomposed,
the illustration is a single blob-like form, but it does have a substantial amount of visual contrast and shading.
